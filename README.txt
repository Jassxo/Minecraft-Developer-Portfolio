Minecraft Developer Portfolio
===============================================

Files:
  index.html  — Main portfolio website (single file, no dependencies needed)

How to use:
  1. Open index.html in any modern browser
  2. To deploy: upload index.html to any web host (GitHub Pages, Netlify, etc.)

Sections:
  - Hero (with your Creeper background)
  - About / Introduction
  - Projects (6 project cards)
  - Servers (5 server entries)
  - Contact (links + form)

Customize:
  - Replace name, project details, and contact info inside index.html
  - Search for "blockcraft" to find all placeholder text
  - Update stats (years, projects, players) in the About section
